    公開
    公開ページに「○○ の本棚」「© ○○」として表示されます。未入力ならアカウント名から仮の名前を使います。

    公開ページの静的サイト（HTML）をここへ書き出します。中身は誰でも読めます。空の public リポジトリ（例 bookshelf-public）を作り、bookshelf-sync アプリのアクセス対象に追加してから選んでください。GitHub Pages（Settings → Pages → Source = main）を有効にすると https://<owner>.github.io/<repo>/ で閲覧できます。
    公開ページを Asayake ハブ（hahero 運営）へ直接公開します。GitHub リポジトリは不要で、Google ログインだけで公開 URL が手に入ります。中身は誰でも読めます。
    あなたの Amazon アソシエイト タグを設定すると、公開ページの商品リンクがあなたの収益になります。空にすると広告（アフィリエイトリンク）を付けません。無料プランでは運営（hahero）のタグが付きます。
    公開ページの作成・編集・公開は、左ペインの 「公開」 ボタンから行います。

    公開ページを管理

    公開したい本棚や本を「公開ページ」にまとめ、スタイルを選んで公開します。公開はページごとに行い、公開中のページには公開 URL が表示されます。公開先（自分の GitHub / 共有ハブ）は 設定 → 公開 で選べます。

    無料プランでは、公開ページの商品リンクに運営（Asayake）のアフィリエイト ID が付きます（収益は運営に入ります）。自分の収益にする・広告を付けないようにするには Plus をご利用ください。
