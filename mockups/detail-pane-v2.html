変更点 (v1 → v2):
  • 並び替え対応: 各セクションが共通の d-section カードに統一、編集モード時に左に grip-vertical ハンドルが出る → 縦ドラッグで並び替え
  • 均一ボタンサイズ: 全 d-btn が height: 32px (small は 26px) で統一、Amazon/編集アクション/Edit toggle 全て同じ規格
  • スタイリッシュなフォーム: d-textarea / d-input / d-select の border + focus 状態 (accent-bg 3px ring) + 角丸 6px で統一
  • セクション階層: 各セクションが薄いカード (--bg 背景) として分離、編集時は --panel 白に切替 → 操作対象が一目で分かる
  • 「基本情報」セクション化: 旧フローティング meta-row → カード内に grid-table 化、メタ編集の <details> も同セクション内に
  • 並び順サンプル: 基本情報 → 所属本棚 → 短文メモ → 長文メモ (ユーザは並び替え可能)
  • edit toggle: 上部 topbar の右端、ラベルと同じ行に置いて title 行の混雑を解消
  • cover 固定: 140px 幅 (旧 160px から少し縮小)、aspect-ratio 2/3 で aligned
判断保留: セクション並び替えの永続化 (本ごと / 全本共通 / どちらも) は要確認
