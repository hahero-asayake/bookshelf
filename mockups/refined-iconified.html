bookshelf — Lucide アイコン版

refined.html の全絵文字を Lucide 線画 SVG に置換。CDN ロード (unpkg)。

差し替え方針
  • 絵文字 → <i data-lucide="名前"></i> に置換
  • サイズ・色は親要素の font-size / currentColor に追従
  • 本棚カテゴリ用アイコンは library / star / briefcase / book-open / flask-conical / code-2 / wrench / book 等で例示
  • 本番統合時は CDN ではなく必要分の SVG を js/icons.js に inline 化する想定

本棚カテゴリのアイコン

  • ユーザが本棚作成時に Lucide アイコン名を選ぶ（または絵文字も併用可）
  • 選択 UI は IconPicker (input + 候補グリッド + 検索)
  • 未選択時は folder をデフォルトに

本番統合時の方式

  • CDN ではなく 必要分の SVG を js/icons.js に inline 化
  • 使うアイコン (約 25 種) なら 5–8 KB 程度
  • ヘルパー: icon('home', { size: 16, class: '...' }) → SVG 文字列を返す
