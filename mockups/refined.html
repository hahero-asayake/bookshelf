折りたたみの操作

  • 左ペイン上部の ‹ をクリックで最小化、最小化ストリップの › で復帰
  • 右ペインも同じく › / ‹
  • キー: ⌘[ 左トグル / ⌘] 右トグル / ⌘\ 両方トグル
  • 状態は settings.paneState: { left, right } に保存（永続化）

最小化ストリップの中身

  • 左: 復帰ボタン + よく開く本棚アイコン（最大 6 個）= 1クリックジャンプ
  • 右: 復帰ボタン + 縦書きラベル「本詳細」
  • 本棚アイコンはピン留め可、または「最近開いた本棚」自動表示

編集モードの所作

  • 各ウィジェット左上の ⋮⋮ ハンドルでドラッグ並べ替え
  • 右上 × で外す（追加リストに戻る、削除ではない）
  • 末尾 ＋ から候補をピック

レイアウトデータ

  • private/main.json の home.widgets[] に保存 
    {
  "home": {
    "widgets": [
      { "id": "stat-total",   "span": 3 },
      { "id": "stat-shelves", "span": 3 },
      { "id": "recent-books", "span": 8, "config": { "limit": 8 } },
      { "id": "today-pick",   "span": 4 },
      { "id": "heatmap",      "span": 8, "config": { "weeks": 26 } }
    ]
  }
}
  • 同期フォルダ経由で複数端末共有可
  • プラグインがウィジェットを追加することも可能（拡張余地）

標準体験の流れ

  • 起動 → 🏠 ホーム（前回開いていた画面を復元）
  • 本棚を開く → 左ツリーから選択、または ⌘K で名前検索
  • 整理する → 中央で本を選び、右ペインでメモ・評価をその場で編集
  • 集中する → 左右どちらも、または両方を最小化して中央に集中
  • 取込・公開・除外等 → ⚙️ 設定モーダル または ⌘K から
  • 振り返る → 🏠 ホームに戻り、ウィジェットで自分の関心に最適化

※ スマホ向け / 公開モードは別途。
